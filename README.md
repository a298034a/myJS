## Homework

### Bingo Number

內定一個數字結果 => let bingoNum = 99<br>
彈出輸入視窗，輸入數字<br>
最多能夠猜3次<br>

三次內答對，顯示結果如下：
```
第1次：11
第2次：99
答對了
```

三次內答錯，顯示結果如下：
```
第1次：11
第2次：88
第3次：66
挑戰失敗
```
